## 🚀 Featured Projects

### 🌾 Agriculture Automation System using LoRa Technology &nbsp;<sub>*Final Year Project — 2026*</sub>
> A complete **off-grid smart-farming ecosystem** with two wireless nodes communicating over LoRa for long-range coverage.

- 🛰️ **Farm Node** (ESP32 + RYLR998 LoRa): auto-controls Peltier AC, humidifier, exhaust fan & irrigation pump via relays, driven by real-time **DHT22, MQ135 & capacitive soil-moisture** readings
- 📟 **Portable Control Node** (ESP32 + LoRa + 1.3" OLED + 5 buttons): 4-page UI for live telemetry, Auto/Manual switching & direct actuator override; data refreshed every 10 s
- 💧 **Smart Water Tank**: custom overflow-prevention circuit (BC547 + resistors + LED indicators + relay)
- ⚡ **Power System**: 12V/20A SMPS + dual 18650 Li-ion + 6V/500mA solar + TP4056 for 24/7 off-grid operation
- ☁️ **Cloud**: MQTT dashboard for remote monitoring & control over a Wi-Fi gateway
