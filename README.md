### 🚀 Featured Projects

#### 🌾 Agriculture Automation System using LoRa Technology *(Final Year Project — Completed 2026)*
> A complete off-grid smart farming ecosystem with two wireless nodes communicating over LoRa for up to 10km range

- **Farm Node** (ESP32 + RYLR998 LoRa): Auto-controls Peltier AC, humidifier, exhaust fan & irrigation pump via relay modules based on real-time DHT22, MQ135 & capacitive soil moisture sensor readings
- **Portable Control Node** (ESP32 + LoRa + 1.3" OLED + 5 buttons): 4-page UI for live telemetry, Auto/Manual mode switching, and direct actuator override; data refreshed every 10 seconds over LoRa
- **Smart Water Tank**: Custom overflow-prevention circuit using BC547 transistors, resistors, LED indicators & relay
- **Power System**: 12V/20A SMPS for actuators + dual 18650 Li-ion + 6V/500mA solar panel + TP4056 for 24/7 off-grid controller operation
- **Cloud**: MQTT dashboard for remote monitoring and actuator control via Wi-Fi gateway
